# SpamMo ‚Äî Spam Email Classifier

Logistic regression spam classifier trained on **8,348 emails** using feature engineering and a TF-IDF-inspired word scoring scheme. Achieved **93.8% training accuracy** and **91% test accuracy**, validated with 10-fold cross-validation.

## Approach

Feature engineering is split into two parts:

**Word features (800 binary indicators)** ‚Äî Each word in the training corpus is scored by how much more frequently it appears in spam versus ham emails, using Laplace-smoothed frequency ratios. The top 800 words by this score become binary presence/absence columns.

**Numeric features (4 columns)** ‚Äî Character count, word count, exclamation mark count, and capitalisation ratio. EDA (see the correlation heatmap in the notebook) showed that exclamation count is the strongest individual signal (~0.3 correlation with the spam label), while word count and character count are near-redundant (~0.95 inter-correlation).

The model is logistic regression with the `liblinear` solver and L2 regularisation.

## Results

| Metric                       | Score |
|------------------------------|-------|
| Training accuracy            | 93.8% |
| Test accuracy (held-out set) | 91.0% |
| 10-fold CV mean              | ~88.9% |

## Dataset

The dataset contains **8,348 labelled emails** (train) and **1,000 unlabelled emails** (test), distributed as a zip archive with two CSVs:

| File        | Columns                          |
|-------------|----------------------------------|
| `train.csv` | `id`, `subject`, `email`, `spam` |
| `test.csv`  | `id`, `subject`, `email`         |

Place `spam_ham_data.zip` in `data/raw/` before running the notebook.

## Getting Started

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Place spam_ham_data.zip in data/raw/

# 3. Launch the notebook from the repo root
jupyter notebook notebooks/spam_classifier.ipynb
```

Running all cells will train the model, save it to `models/spam_classifier.joblib`, write figures to `results/figures/`, and export predictions to `results/spam_predictions.csv`.
